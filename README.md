# 👟 StepClean Taubaté

Landing page premium desenvolvida para a **StepClean Taubaté**, focada na prestação de serviços de limpeza e restauração profissional de tênis. O projeto utiliza tecnologias de ponta para criar uma experiência visual imersiva e responsiva.

## 🔗 Link do Projeto
🚀 [Acesse o site oficial aqui](https://stephaniebuilds.github.io/stepclean-taubate/)

## ✨ Funcionalidades Principais

* **Visual Imersivo (WebGL):** Fundo animado utilizando *shaders* customizados que reagem ao movimento do cursor.
* **Exibição 3D:** Integração com **Three.js** para exibição de modelos de tênis, com transição suave entre texturas ("Antes" e "Depois").
* **Interface Moderna:** Design aplicando *Glassmorphism* e *Tailwind CSS* para uma estética limpa, sofisticada e profissional.
* **Componentes Interativos:**
    * Modal de galeria de resultados (Fotos/Vídeos).
    * Menu responsivo otimizado para dispositivos móveis.
    * Botões de ação rápida para agendamento direto via WhatsApp.
* **Responsividade:** Layout 100% fluido, adaptando-se perfeitamente de dispositivos mobile a desktops de alta resolução.

## 🛠️ Tecnologias e Ferramentas

O projeto foi construído priorizando a performance gráfica e a experiência do usuário (UX):

* **Front-end:** HTML5, Tailwind CSS (plugins de formulários e container-queries).
* **JavaScript:** ES6+ para manipulação de DOM, estados de modais e lógica de navegação.
* **Gráficos 3D:** Three.js (r125) e WebGL para renderização de texturas e animações customizadas.
* **Tipografia:** Google Fonts (Manrope e Be Vietnam Pro).
* **Ícones:** Material Symbols.
